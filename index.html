A Arquitetura de Dados

Este Dashboard é gerado estaticamente por um Motor em Python (Pandas) ultrarrápido para suportar o cruzamento de milhares de lutas simultâneas sem travar o seu navegador.

Por conta disso, a atualização dos dados não é feita enviando arquivos pelo navegador, mas sim atualizando os arquivos fontes locais.

Instruções de Atualização:

  1. Extraia o seu novo Battle Record.json do jogo.
  2. Substitua o arquivo antigo na pasta:
    data/raw/
  3. Abra seu terminal e execute o gerador:
    python src/gerador_relatorios.py
  4. Pronto! Dê F5 nesta página e os dados estarão atualizados.
